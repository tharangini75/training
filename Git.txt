Git:
Git is a distributed version control system used to track changes in source code, manage different versions, and enable multiple developers to work on a project efficiently. local repository

GitHub is a web-based hosting platform that uses Git to store repositories online and helps developers share code, collaborate, and manage projects. remote repository
